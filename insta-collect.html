insta-collect

A modular automation tool for Instagram data extraction, optimized for research and social analysis.

View on PyPI

Installation

pip install insta-collect
Note: Playwright binaries required: 
playwright install

CLI Usage

insta-collect --tag donaldtrump --limit 7
