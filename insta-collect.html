instacollect

A modular automation tool for Instagram data extraction, designed for evidence-based research and social analysis.

View on PyPI

Installation

Install the package via PyPI or directly from source:

pip install insta-collect
pip install git+https://github.com/Alammahadika/insta-collect.git
Note: Playwright binaries are required for browser automation. 
playwright install

CLI Usage

Run the scraper via terminal to collect photo-based posts by hashtag:

insta-collect --tag donaldtrump --limit 7
